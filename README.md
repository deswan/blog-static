# blog-static

> Nuxt & markdown 驱动的静态博客生成器

## Build Setup

``` bash
# install dependencies
$ npm install

# 开发
$ npm run dev

# 构建
$ npm run build
$ npm start

# 生成静态博客
$ npm run server
$ npm run generate

# git一键部署
# 请按照 deploy/condig.sample.js 创建 deploy/config.js
$ npm run deploy
```
